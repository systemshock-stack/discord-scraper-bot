# Discord Scraper Bot

A **clean, safe Discord bot** that:
- Listens to multiple channels
- Saves clean messages to a file
- Runs 24/7

**No token. No data. No server info.**

## Features
- Real-time message capture
- Removes URLs, @mentions, bots
- Weekly auto-post
- Idle history scraping
